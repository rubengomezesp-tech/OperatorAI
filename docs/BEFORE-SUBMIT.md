# Before Submitting to App Store

## 1. Create Review Account
Apple reviewers need a test account. Create one in your app:
- Email: review@operatorai.app
- Password: OperatorReview2026!
- Make sure this account has a Pro plan active (or free trial)

## 2. Take Screenshots
On your iPhone (or Simulator):
- Screenshot 1: Landing page / Login
- Screenshot 2: Dashboard
- Screenshot 3: Chat with an AI-generated image visible
- Screenshot 4: Image Studio with gallery
- Screenshot 5: Billing page with plans

Save as PNG, 1290x2796 for 6.7" and 1179x2556 for 6.1"

## 3. Verify All URLs
Open each in Safari and confirm they work:
- https://www.operatoraiapp.com/privacy
- https://www.operatoraiapp.com/terms
- https://www.operatoraiapp.com/support
- https://www.operatoraiapp.com/delete-account

## 4. Test on Device
- Install via Xcode on your iPhone
- Go through full flow: signup → dashboard → chat → generate image → billing
- Toggle language ES/EN
- Check all pages load without errors

## 5. Submit
- Xcode: Product → Archive → Distribute → App Store Connect
- App Store Connect: fill in all fields from APP-STORE-GUIDE.md
- Submit for review
- Wait 24-48 hours for Apple review
